# OOP Arena Combat Simulator 🛡️

A robust Object-Oriented Programming (OOP) simulation built in C# that replicates the core mechanics, tactical roles, and support structures of a class-based arena game.

**Kullanılan Temel C# / OOP Kavramları:**
* **Class Management & Archetypes:** Karakter sınıflarının ve taktiksel rollerin (Örn: Defender sınıfının alan savunması ve defansif destek mekanikleri) algoritmik olarak ayrıştırılması.
* **Inheritance (Kalıtım):** Farklı karakter tiplerinin temel bir `Karakter` sınıfından miras alarak kendi özel yeteneklerini ve temel statülerini (Can, Hasar, Defans) inşa etmesi.
* **Complex Object Interaction:** Savaş alanı simülasyonunda nesnelerin (karakterlerin) birbirleriyle olan etkileşimlerinin, hasar hesaplamalarının ve hayatta kalma mekaniklerinin C# mimarisiyle koda dökülmesi.
